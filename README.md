# 🏠 PG Hostel Management System

A full-stack web application to manage a 
PG hostel with 15 rooms across 3 floors.

## Problem Solved
Hostel owner was manually checking each room
to track payments. This app digitizes everything.

## Features
- ✅ Secure admin login
- ✅ Visual room dashboard (15 rooms, 3 floors)
- ✅ 4 members per room with individual tracking
- ✅ Payment status — Paid / Unpaid / Overdue
- ✅ Move-in date and payment date tracking
- ✅ Search and filter tenants
- ✅ MySQL database with PHP REST API

## Tech Stack
| Frontend  | Backend | Database | Server |
|-----------|---------|----------|--------|
| HTML, CSS | PHP     | MySQL    | XAMPP  |
| JavaScript|         |          |        |

## Pages
1. Login page
2. Dashboard — room grid
3. Tenants — member list
4. Payments — payment tracking

## How to Run
1. Install XAMPP
2. Start Apache and MySQL
3. Import database SQL file
4. Open localhost/pg-hostel2
